== Description ==

This plugin allows you to publish a wishlist on your blog - With links to the items so your users can see them!

You simply add a list of items you want to add to your wishlist, enable the widget and your items will be displayed. Once you've received an item, you can click "Done!" and then the item will have a line crossed through it to show that you've now got that item!

== Installation ==

1. Upload `jr-wishlist.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit the JR Wishlist Settings in your Administration Panel, Settings --> JR Wishlist.
4. Enable the Widget through the Appearance --> Widgets tab.
